Live demo of AES-256-GCM encryption running entirely in your browser via the Web Crypto API — no data is sent to any server.
Uses the same algorithm as the original Java desktop app: PBKDF2-HMAC-SHA256 key derivation → AES-256-GCM authenticated encryption.

Key derivation: PBKDF2 with HMAC-SHA-256, 100,000 iterations, 256-bit key.
Encryption: AES-256-GCM with a random 12-byte IV prepended to the ciphertext.
Output format: Base64( IV[12 bytes] + ciphertext + GCM auth tag[16 bytes] )
Salt is fixed (matching original Java implementation) — not recommended for production use.
